# How to run this project

It is strongly recommended that you download the compressed package of the entire project or use the `git clone` command to pull the entire project.

Then, enter the project folder, click the `index.html` file, and open it with a browser.

Although the static webpage has been uploaded to GitHub Pages `(https://bugs-maker-richard.github.io/logistics_assignment_April/)`, the browsing effect is not satisfactory based on testing under certain network environment.. Therefore, it is strongly suggest that you run it locally.
